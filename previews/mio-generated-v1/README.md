# みおちゃん GIFアニメーション候補 v1

システム未反映の確認用セットです。組み込み前に `index.html` で動きを確認してください。

## GIF一覧

| ファイル | 状態 | 想定用途 |
|---|---|---|
| `anxious.gif` | 不安・緊張 | ゲーム開始／困り度が高いとき |
| `overwhelmed.gif` | 仕事を抱えすぎ | タスク過多の設問 |
| `thinking.gif` | 考え中・混乱 | 回答を考えているとき |
| `relieved.gif` | 安心・ほっとする | 良いアドバイスを受けたとき |
| `presentation.gif` | 発表の練習 | プレゼン関連の設問 |
| `time-pressure.gif` | 時間切迫 | 残り時間が少ないとき |
| `idea.gif` | ひらめき | 具体的な行動案を理解したとき |
| `celebrate.gif` | 成功・お祝い | 高得点／ゲームクリア |
| `listening.gif` | 傾聴 | 自由記述を入力・評価中 |
| `retry.gif` | やさしく再挑戦 | 低品質・無効回答のとき |

各GIFは384×512ピクセル、8フレーム、透過背景、無限ループです。

## 生成プロンプト方針

- 既存の `spritesheet.webp` と `idle.gif` をキャラクター参照として使用
- 同じ髪型、髪留め、服装、IDストラップ、体型、ちびキャラ表現を維持
- 状態ごとに4列×2行の8フレームスプライトシートを生成
- 単色グリーン背景で生成後、クロマキー除去して透過GIFへ変換
- 文字、ロゴ、透かし、余分な人物、背景、影は追加しない

Built-in image generationを使用しています。生成元シートは `source-sheets/` に保存しています。
