# みおちゃんアニメーションアセット

正式なみおちゃんGIF 9点を配置済みです。添付スプライトシートから2倍解像度で再構成し、全ファイルを`384×416`、透過GIF、無限ループに統一しています。

原本はユーザーから提供された添付ファイルです。

画面状態との対応、再生ルール、アクセシビリティ要件は[仕様書](../../DEMO_SPEC.md#7-アニメーション仕様)を参照してください。

## 生成アニメーション v1

承認済みのシステム専用アニメーションは`generated-v1/`に保存しています。各ファイルは`384×512`、8フレーム、透過GIF、無限ループです。

| ファイル名 | 使用場面 |
|---|---|
| `anxious.gif` | 困り度が高い場面、失敗や緊張の設問 |
| `overwhelmed.gif` | 仕事を抱えすぎた設問 |
| `thinking.gif` | 回答確認、DB内Embedding、採点中 |
| `relieved.gif` | 安心した反応、通常の結果発表 |
| `presentation.gif` | 発表・説明に関する設問 |
| `time-pressure.gif` | 残り10秒 |
| `idea.gif` | 成長・ひらめきに関する設問 |
| `celebrate.gif` | A/Bランク、レスキュー成功 |
| `listening.gif` | 初期画面、アンケート、自由記述入力中 |
| `retry.gif` | 無効回答、再挑戦、Eランク |

従来のGIFは切り戻し用として直下に保持しています。
